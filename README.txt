Project MegaPhone

このプロジェクトは、「応援」を共有するファンコミュニティを作る事で、
市民球団を支える新しい仕組みを作る試みです。 
21世紀型の樽募金として、今まで避けられない弱点であった資金面を、
ファンとコミュニティの力で解決することを目指しています。

MegaPhone コンセプトアプリはこちら!
https://m-ph.org/

本件は、ひろしま発人材集積プロジェクト「ヒロハタ」に参加し、事業化を目指しています。 
事業化後に発生する利益は、コミットくださった全ての方に、望まれる形で等しく還元されるとは限りません。
